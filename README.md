# Sales-Dashboard
Interactive Sales Dashboard project built using Excel, MySQL, and Power BI to analyze sales performance, profit, regional trends, and business insights through SQL queries and data visualization.

## Overview

This project focuses on analyzing retail sales data to understand business performance and identify useful insights. The data was cleaned, explored using SQL, and presented through an interactive Power BI dashboard. The dashboard allows users to monitor sales, profit, and regional performance in a simple and effective way.

---

## Objectives

- Prepare and clean the sales dataset.
- Analyze sales data using SQL.
- Create an interactive Power BI dashboard.
- Track key business metrics.
- Support data-driven decision-making.

---

## Tools Used

- Microsoft Excel
- MySQL
- Power BI

---

## Dataset Information

The project uses the **Sample Superstore** dataset, which contains information about:

- Sales
- Profit
- Quantity
- Category
- Sub-Category
- Region
- State
- City
- Segment
- Discount

---

## Dashboard Features

- Total Sales
- Total Profit
- Total Quantity Sold
- Sales by Category
- Sales by Region
- Sales by Segment
- Top Performing States
- Interactive Filters

---

## SQL Analysis

The SQL queries in this project were used to:

- Count total records
- Calculate total sales
- Calculate total profit
- Find average sales
- Compare sales across categories
- Compare sales across regions
- Identify top cities by sales
- Identify top states by profit
- Calculate average discounts
- Find the highest-selling sub-categories
